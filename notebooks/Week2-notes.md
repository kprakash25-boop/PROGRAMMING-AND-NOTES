Week 2 – Variables and Types

Topics
	•	Variables
	•	Data types
	•	Input and output

Key Points
	•	A variable stores data in memory.
	•	Common data types:
	•	int → whole numbers
	•	float → decimal numbers
	•	str → text
	•	input() is used to take data from the user.
	•	print() is used to show output.

Examples:

age = 23
name = "Prakash"
