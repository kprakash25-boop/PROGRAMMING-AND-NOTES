Topics
	•	What is programming
	•	What is Python

Key Points
	•	Programming is writing instructions for a computer to solve a problem.
	•	Python is a high-level programming language that is easy to read and write.
	•	An algorithm is a step-by-step method to solve a problem.
	•	A Python program is written as lines of code that are executed from top to bottom.
